== Description ==

Create sinema on WP.

== Installation ==

The easy way:

1. Go to the Plugins Menu in WordPress
1. Search for "Online Cnema"
1. Click "Install"

The not so easy way:

1. Upload the `online-cinema` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Create new media post.
